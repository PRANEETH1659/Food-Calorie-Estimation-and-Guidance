# Food-Calorie-Estimation-and-Guidance
# 🥗 AI-Based Calorie-Aware Diet Advisor

A deep learning project that detects food from images and provides personalized diet advice using AI.

> VIT-AP University | B.Tech CSE Project

---

## 🚀 Features
- 📸 Upload any food image → instant food detection
- 🔥 Calorie estimation per serving
- 🤖 Personalized diet advice powered by Groq AI (LLaMA 3.3)
- 📊 Macro breakdown (Protein / Carbs / Fats)
- 🍽️ Next meal recommendations
- 📅 Full end-of-day meal plan

---

## 🛠️ Tech Stack
| Component        | Technology              |
|-----------------|-------------------------|
| Food Detection  | TensorFlow / MobileNetV2|
| Dataset         | Food-101 (101 classes)  |
| AI Advice       | Groq API (LLaMA 3.3 70B)|
| UI              | Gradio                  |
| Platform        | Google Colab            |
